# AnalyzeProjects Project Record

AnalyzeProjects is the CTS-governed command tool that evaluates the explicitly registered Aptlantis development portfolio. Its canonical evaluation inventory is `ProjectIndex.md`; its machine-readable identity and lifecycle are recorded in `AnalyzeProjects.manifest.toml`.

The current operating contract is documented in `README.md`. The tool evaluates 30 projects across DRS, CTS, WDS, and internal Standards groups, using environment-only OpenAI authentication and preserving structured JSON compatibility for the dashboard.
